# Poubelle-detection 

Application Streamlit pour détecter automatiquement si une poubelle est vide ou pleine à partir d'une image ou d'une vidéo, grâce au modèle YOLOv8.

## Fonctionnalités
- Détection sur image 
- Détection sur vidéo 
- Affichage direct des résultats annotés
- Téléchargement de la vidéo annotée

## Technologies utilisées
- Python
- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV

## Lancer l'application localement
```bash
streamlit run appPoubelle.py
